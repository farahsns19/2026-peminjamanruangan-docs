# Dokumentasi API Backend

Backend dibangun menggunakan ASP.NET Core Web API.

Base URL:
http://localhost:5133/api/PeminjamanRuangan

---

## Endpoint CRUD

### GET Semua Data
GET /api/PeminjamanRuangan

### GET Detail by ID
GET /api/PeminjamanRuangan/{id}

### POST Tambah Data
POST /api/PeminjamanRuangan


### PUT Update Data
PUT /api/PeminjamanRuangan/{id}

### DELETE Hapus Data
DELETE /api/PeminjamanRuangan/{id}

---

## Status Peminjaman

Field status dapat bernilai:

- Menunggu
- Disetujui
- Ditolak
